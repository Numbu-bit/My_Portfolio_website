Project 2

Smart Attendance System A face-recognition–based attendance system built to automate student check-ins in classrooms. The system uses a webcam to capture images, applies face recognition algorithms to identify registered students, and stores attendance data in a secure database. Tech stack: Python, OpenCV, Flask, SQLite
Key features:

  • Automated face detection and recognition
  • Real-time attendance logging
  • Simple web interface for viewing reports

Impact: Eliminates manual attendance processes and reduces errors while saving time..
